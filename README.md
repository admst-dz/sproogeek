## 📖 О проекте

**Sproogeek 3D** — это инновационный 3D-конфигуратор сувенирной и полиграфической продукции (ежедневники, календари, скетчбуки). Проект объединяет в себе тяжелую энтерпрайз бизнес-логику и бесшовный пользовательский опыт.

Система решает проблему долгого согласования макетов между Клиентом (B2C/B2B), Дилером и Производством, перенося весь процесс — от кастомизации изделия до генерации Print-Ready PDF-файлов — в единую онлайн-экосистему.

> ⚠️ **Текущий статус: MVP (Minimum Viable Product)**  
> Проект находится на стадии активной обкатки бизнес-гипотез и архитектуры. Внедрены базовые модели (Твердый переплет, Пружина) и основные роли пользователей.

## ✨ Ключевые возможности

### 🎨 Для Клиентов (B2C & B2B)
*   **Real-time 3D Визуализация:** Полная кастомизация продукта прямо в браузере (выбор переплета, цвета обложки, пружины, резинки).
*   **Тиснение и Брендирование:** Загрузка собственного логотипа с мгновенным отображением эффекта тиснения на 3D-модели.
*   **Система Токенов (ТК):** Корпоративные клиенты могут выдавать токены сотрудникам для самостоятельного заказа брендированной продукции в рамках лимита.
*   **Онлайн-согласование:** Автоматическая генерация рендеров для утверждения макета без участия менеджера.

### 💼 Для Дилеров и Производства
*   **CRM и Дашборд Дилера:** Управление заказами, эмуляция заказов за клиента, генерация Коммерческих Предложений (КП).
*   **Server-Side Rendering (SSR):** "Безголовый" (Headless) микросервис автоматически создает фотореалистичные скриншоты 3D-сцен для спецификаций.
*   **Авто-генерация Техкарт:** Микросервис формирует PDF-документ (Технологическую карту) для типографии на основе заказа.

## 🏗 Архитектура и Стек технологий

Проект построен на базе **Event-Driven Microservices** в рамках монорепозитория, что обеспечивает высочайшую отказоустойчивость и скорость работы.

### 🌐 Frontend (Клиент)
*   **React + Vite:** Быстрая сборка и HMR.
*   **Three.js + React Three Fiber (R3F):** Отрисовка 3D-графики, работа с `.glb` моделями.
*   **Zustand:** Легковесное и быстрое управление состоянием (State Management).
*   **TailwindCSS:** Стилизация интерфейса (Glassmorphism, адаптивность).

### ⚙️ Backend (Ядро)
*   **FastAPI (Python 3.10+):** Высокопроизводительный асинхронный REST API.
*   **PostgreSQL + asyncpg:** Надежное хранение реляционных данных и JSON-конфигураций.
*   **SQLAlchemy 2.0 + Alembic:** Асинхронная ORM и контроль миграций БД.
*   **JWT & OAuth2:** Собственный сервер авторизации (Google, Telegram, Email).

### 🧩 Микросервисы (Воркеры)
*   **Render Service (Node.js + Puppeteer):** Захват 3D-рендеров на стороне сервера (Chrome Headless WebGL).
*   **PDF Generator (Python + WeasyPrint):** Преобразование HTML-шаблонов с данными заказа в PDF-техкарты.
*   **Apache Kafka + Zookeeper:** Шина событий для асинхронной связи между API и тяжелыми фоновыми задачами.

### 🐳 DevOps & Инфраструктура
*   **Docker & Docker Compose:** Полная изоляция сервисов.
*   **Gunicorn:** Многопоточная обработка запросов на бэкенде.
*   **Traefik:** Автоматическое получение SSL сертификатов (Let's Encrypt) и reverse-proxy.
*   **GitHub Actions + Watchtower:** Автоматический CI/CD пайплайн (сборка образов -> деплой на сервер).
*   **Sentry:** Мониторинг ошибок и логирование.
